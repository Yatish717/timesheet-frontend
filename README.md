# Timesheet Frontend (AdAppr)

This project is a web-based timesheet and attendance management application built with React, using Create React App as its foundation. It features user authentication, dashboards, attendance tracking, leave management, and more, with a modern UI leveraging Ant Design, Material UI, Bootstrap, and Tailwind CSS.

## Features
- User authentication (login, password reset)
- Dashboard for employees and managers
- Attendance and leave management
- Project and cost center tracking
- User profile management
- Data tables and reporting
- Responsive design with modern UI libraries

## Technologies Used
- React 18
- React Router
- Ant Design, Material UI, Bootstrap, Tailwind CSS
- Axios for API requests
- Day.js, Moment.js, date-fns for date handling
- PrimeReact, Mobiscroll, Syncfusion for advanced UI components
- Jest and React Testing Library for testing

## Folder Structure
- `src/Pages/` — Main application pages (Dashboard, Login, Attendance, Leave, etc.)
- `src/` — Entry points, global styles, and utilities
- `public/` — Static assets and HTML template
- `stylesheets/` — Custom CSS (e.g., 3D animation)

## Setup & Installation
1. **Install dependencies:**
	```bash
	npm install
	```
2. **Start the development server:**
	```bash
	npm start
	```
	The app will be available at [http://localhost:3000](http://localhost:3000).

3. **Run tests:**
	```bash
	npm test
	```

4. **Build for production:**
	```bash
	npm run build
	```

## Scripts
- `npm start` — Start development server
- `npm test` — Run tests
- `npm run build` — Build for production
- `npm run eject` — Eject configuration (not reversible)

## Customization
- UI components can be customized via Ant Design, Material UI, and Tailwind CSS.
- Main entry: `src/index.js`, routes defined with React Router.
- Add or modify pages in `src/Pages/`.
